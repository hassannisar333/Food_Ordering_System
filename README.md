# Food_Ordering_System

## Overview

The Food Ordering System is a console-based application developed in C++ that allows users to browse a menu, place food orders, calculate bills, and manage customer orders efficiently. The project was designed to apply Object-Oriented Programming (OOP) concepts and improve problem-solving and software development skills.

## Features

* Display food menu and prices
* Place customer orders
* Add multiple food items to an order
* Calculate the total bill automatically
* Generate an order summary
* User-friendly console interface
* Modular and maintainable code structure

## Technologies Used

* Programming Language: C++
* Concepts: Object-Oriented Programming (OOP)
* Development Environment: Visual Studio Code / Dev-C++

## OOP Concepts Implemented

* Classes and Objects
* Encapsulation
* Inheritance
* Polymorphism
* Functions and Data Abstraction

## Project Structure

* Menu Management
* Order Processing
* Billing System
* Customer Interaction Module

## How It Works

1. The user starts the application.
2. The menu is displayed with available food items and prices.
3. The user selects one or more items.
4. The system calculates the total bill.
5. An order summary and final bill are displayed.
